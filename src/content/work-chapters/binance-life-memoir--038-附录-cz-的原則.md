<p id="page_226" class="calibre3"><span class="calibre4">附录：CZ 的原则</span></p><p class="calibre1"><span class="calibre7">这是一个持续更新的文档，记录了我的一些原则和做事方式。不一定适合所有人。</span></p><p class="calibre1"><span class="calibre10">思维方式</span></p><p class="calibre1">1. 别浪费时间</p><p class="calibre1"><span class="calibre7">时间是最有限的资源，要用好。</span></p><p class="calibre1"><span class="calibre7">管好时间，关键不是做什么，而是不做什么。</span></p><p class="calibre1"><span class="calibre7">你不用列待办清单，重要的事自己会冒出来。</span></p><p class="calibre1"><span class="calibre7">你得有个「不做清单」，不然时间不知不觉就被耗掉了。</span> <span class="calibre7">不开长会，不搞「认识一下」，「探索合作」这种介绍会。人类 80% 的沟通都是八卦，直接砍</span> <span class="calibre7">掉。</span></p><p class="calibre1"><span class="calibre7">别说废话，不用说「我长话短说」，直接长话短说就行。</span> <span class="calibre7">别寒暄。</span></p><p class="calibre1"><span class="calibre7">有话直说，直奔主题。</span></p><p class="calibre1"><span class="calibre7">每个会都要有明确的目的、你要什么？</span></p><p class="calibre1"><span class="calibre7">会默认就开五分钟。</span></p><p class="calibre1"><span class="calibre7">别写长文，别做 PPT。</span></p><p class="calibre1"><span class="calibre7">列几条要点，画个柱状图，够了（就像这份文档）。</span></p><p class="calibre1"><span class="calibre7">时间这话题和好多其他原则都有关。</span></p><p class="calibre1"><span class="calibre7">别追那种听著高大上的合作，捡容易的做。</span></p><p class="calibre1"><span class="calibre7">早拒绝，两边都省时间。</span></p><p class="calibre1"><span class="calibre7">我做业务是被动型的，人家来找我，反而成事快。</span></p><p class="calibre1"><span class="calibre7">少社交。</span></p><p class="calibre1"><span class="calibre7">朋友重质不重量。</span></p><p class="calibre1"><span class="calibre7">我不看电视、新闻、体育，也不听歌。</span></p><p class="calibre1"><span class="calibre7">我抠时间抠到有人说我没礼貌。没事，我要的不是礼貌，是效率。</span> <span class="calibre7">时间比钱紧多了。你真开始惜时，成功和钱自然就来了。</span> <span class="calibre7">小变化，时间长了，差别巨大。</span></p><p class="calibre1">2. 别只盯著钱</p><p class="calibre1"><span class="calibre7">太多人死命追钱，反而「得不到」。</span></p><p class="calibre1"><span class="calibre7">别追钱，要创造价值。你能创造 10 分价值，只拿 8 分，一直这么干，钱多到你不敢想。要是</span> <span class="calibre7">你只拿 5 分，还能做大，那钱来得更快。</span></p><p class="calibre1"><span class="calibre7">赚钱不是一笔买卖挣最多，而是每次留点甜头，人家才老找你，长期下来你才挣得多。</span> 3. 让有限的资源发挥出价值</p><p class="calibre1"><span class="calibre7">世上很多东西都有限：健康、精力、人（团队、朋友、家人）、时间。很多都难扩大。</span> <span class="calibre7">你只能把时间和健康省著用。</span></p><p class="calibre1"><span class="calibre7">人这块，靠建强队来放大，这通常也是领导最值钱的本事。</span> <span class="calibre7">朋友呢，深交不了几个。我不是人脉中心，也维持不了太多关系，太费时间。我会保留几个</span> <span class="calibre7">「枢纽型」朋友，他们认识人多，我哪天要「搭线」了，找他们就行。</span> <span class="calibre7">你可能注意到了，钱没在上面。钱不是有限资源，别太当回事。把信誉立起来，钱自然好拿。</span> 4. 守住底线</p><p id="page_227" class="calibre1"><span class="calibre7">道德红线，碰都别碰。碰了早晚还回来。</span></p><p class="calibre1"><span class="calibre7">对待用户，永远做对的事，不是做容易的。保护好用户，你成功（和有钱）得超乎想象。</span> <span class="calibre7">5. </span><span class="calibre22">公</span><span class="calibre22">平待</span><span class="calibre22">人</span></p><p class="calibre1"><span class="calibre7">这条覆盖很宽，管个人关系、同事、生意。我真信公平待人。别占人便宜，也别让人占你便</span> <span class="calibre7">宜。</span></p><p class="calibre1"><span class="calibre7">说来容易做来难。人嘛，都带点主观，都偏向自己。你得忍著，尽量客观。反过来，碰到「强</span> <span class="calibre7">势」的，又别太软。</span></p><p class="calibre1"><span class="calibre7">这个平衡拿不准，走不远。所以，得学会。</span></p><p class="calibre1">6. 搞长期、双赢的关系或合作</p><p class="calibre1"><span class="calibre7">「长期」「双赢」都被说烂了。但我信这个。成功是长期攒出来的，要维持健康的长期关系，</span> <span class="calibre7">就得搞双赢。一笔买卖要合理，就得两边都得好处。永远要问，对方图啥。</span> <span class="calibre7">一头热的买卖长不了，你总得找（弱的）新伙伴，那投入产出比是负的。短期的赢，一般都</span> <span class="calibre7">小。我反对短期赢，它会抹杀长期赢，让你分心，没法盯长期。短期项目本身也耗时间，隐藏</span> <span class="calibre7">的机会成本高。</span></p><p class="calibre1"><span class="calibre7">盯长期，搞大的赢。</span></p><p class="calibre1"><span class="calibre7">玩「无限的游戏」（看 Simon Sinek 的书《无限的游戏》）。</span> <span class="calibre7">我这说的「短期赢」，是指一次性那种，不是指一步步累积、最后成大事的小胜利。后者非常</span> <span class="calibre7">需要。</span></p><p class="calibre1">7. 专注</p><p class="calibre1"><span class="calibre7">成功不看你做多少事，看你选的很少的几件事里，干得有多好。专注才能让你往深里干。把干</span> <span class="calibre7">扰清掉。</span></p><p class="calibre1"><span class="calibre7">我没什么爱好。每天运动 30 分钟。也没多少要打理的东西。哪些东西时间成本高。</span> <span class="calibre7">除了币安，我很少干别的 (这条写于 2022 年)。不像有些特成功的人，好几家公司的 CEO。我</span> <span class="calibre7">基本不碰别的生意，也不搞分散理财（对多数人这可能不好，但我管用）。</span> <span class="calibre7">这么做的风险是可能太窄，错过机会。但今天这信息爆炸的世界，那很难发生。</span> 8. 远离「有毒」的关系</p><p class="calibre1"><span class="calibre7">有不道德的、难缠的，有跟你价值观、使命不合的，还有特耗你时间的。让这些人走，清出你</span> <span class="calibre7">生活。最后这种「耗时间」的，最容易忽略。</span></p><p class="calibre1"><span class="calibre7">还有一种「维护成本高」的人，老要你关注，为小事不爽，要找你聊，要你反复夸他重要。这</span> <span class="calibre7">些人其实没错，但我（CZ）伺候不了。我只跟「省心」、自信的人打交道。</span> <span class="calibre7">还有一种「话痨」，特别喜欢聊天的。这也没错。但我没空闲聊。这种我也躲著。后面聊沟通</span> <span class="calibre7">还会说。</span></p><p class="calibre1">9. 正面心态</p><p class="calibre1"><span class="calibre7">生活里谁都有坎儿，家人、朋友、工作。正面积极的心态，往往会导致结果好。具体为啥我也</span> <span class="calibre7">说不准，但世界就这么转的。</span></p><p class="calibre1"><span class="calibre7">有句话我记了多年，对我一直很有帮助。</span></p><p id="page_228" class="calibre1"><span class="calibre7">问：「人生走到谷底怎么办？」</span></p><p class="calibre1"><span class="calibre7">答：「继续走，就能出来。」</span></p><p class="calibre1"><span class="calibre7">还有，你知道自己干的是对的事，那些负面的就淡了。你会有股正劲儿。这也是守底线的一大</span> <span class="calibre7">好处。</span></p><p class="calibre1"><span class="calibre7">10. </span><span class="calibre22">自</span><span class="calibre22">己扛</span><span class="calibre22">事</span></p><p class="calibre1"><span class="calibre7">心态对了，是成功的前提。对自己干的活负责，还得有自豪感。</span> <span class="calibre7">别光「做任务」，要「当自己的事」。别光砌墙，要想着你在盖教堂。要是你不觉得在「盖教</span> <span class="calibre7">堂」，那你该换个活儿干。</span></p><p class="calibre1"><span class="calibre7">想想还有啥没人说、但你能做的，让它更好。对结果负责，成了败了都担着。你真有这心态，</span> <span class="calibre7">活儿会干得出奇好，你往上走也快。</span></p><p class="calibre1">11. 不停地学</p><p class="calibre1"><span class="calibre7">世界在变，你得跟着学。读书，去不同城市住住，建立世界观。</span> <span class="calibre7">我不怎么看新闻和短文。觉得看十分钟的文章，容易点头，但想不深，看完就忘。</span> <span class="calibre7">书讲得深，有道理、有例子。一口气读上八小时，有些新东西能留下。我觉得书是自我提升回</span> <span class="calibre7">报率最高的。我买很多，一年三百本左右，但不都读完。一年能完整读完大概八十本。</span> <span class="calibre7">我每天试着写点东西，博客或文章（像这篇）。写东西帮我理清思路，让想法更顺。写作本身</span> <span class="calibre7">就是在思考。</span></p><p class="calibre1">12. 做个早期采用者</p><p class="calibre1"><span class="calibre7">新东西我愿试。下新应用、新工具，常推动币安也换。我读书，也老把新原则往币安里试。这</span> <span class="calibre7">也是我早进区块链的原因，它是跟钱有关的新技术。</span></p><p class="calibre1"><span class="calibre7">早期采用的人常拿指数级回报，同时只需应对有限的风险。只要你把下行风险控住，那些指数</span> <span class="calibre7">级回报就能兑现。做早期的人。</span></p><p class="calibre1"><span class="calibre7">谁都在某方面是早期的人。比如你爱一上映就看电影，那你就是电影尝鲜的。我一般等朋友说</span> <span class="calibre7">好才看。显然，看电影早对你生活没多大用。挑有影响的领域，去那儿当早期的人。</span> 13. 看懂这个世界</p><p class="calibre1"><span class="calibre7">你想成点事，得懂世界怎么转。</span></p><p class="calibre1"><span class="calibre7">人怎么转，可以读《人类简史》。我们活在人造的世界。每个人脑子里的世界都不一样，等于</span> <span class="calibre7">活在不同世界。你得懂他们的世界。</span></p><p class="calibre1"><span class="calibre7">别老非黑即白。世界很少是二元的。很多人把世界想太简单，那很难成功。看深点。现实里，</span> <span class="calibre7">多数事是渐变的。找到平衡通常才是关键，比简单的「是/否」管用。</span> <span class="calibre7">比如，有人天真地以为所有规则都好。规则是人定的，不一定都对。可以读读《论法律》(1850</span> <span class="calibre7">年)、《一课经济学》这类书。</span></p><p class="calibre1"><span class="calibre7">质疑世界。世界由那些不认「常识」或「规矩」的人建起来的。对，火箭不一定要用完就扔，</span> <span class="calibre7">你可以让它反过来，着陆回收。汽车就算只能走铺装路，也很有用。地球不是平的。钱不该是</span> <span class="calibre7">无限供应的资产。</span></p><p class="calibre1"><span class="calibre7">也要理解世界的局限。我们不活在完美世界。别打打不赢的仗，绕开，换地儿，干别的。这世</span> <span class="calibre7">界能干的多了，盯你能做的积极贡献。</span></p><p class="calibre1">14. 别被标签糊弄</p><p class="calibre1"><span class="calibre7">人爱贴标签，帮我们理解和适应世界。但标签也常糊弄我们，限制想法。你得看穿标签，去理</span> <span class="calibre7">解第一性原理，就是事情本质是啥，如何运作。</span></p><p id="page_229" class="calibre1"><span class="calibre7">比如，一听「公司」，你可能想到忙碌办公楼忙活的人。但公司是啥？那张注册纸？那个办公</span> <span class="calibre7">室？是人？是产品？现在，一群人能一起干活，没注册公司，没实体办公室，甚至没中央指</span> <span class="calibre7">挥。他们照样高效合作，有合理的激励方案。</span></p><p class="calibre1"><span class="calibre7">再比如，钱是啥？我看过无数关于「什么是钱」的争论。「政府发的才是钱」，「不能付帐单</span> <span class="calibre7">就不是钱」，「比特币不是钱」。好吧，我根本不关心比特币能不能叫「钱」。我关心它是否</span> <span class="calibre7">有用、有价值、有流动性。</span></p><p class="calibre1"><span class="calibre7">我们聊的 99% 的东西都是标签。职位、排名、公司，甚至国家，都是人发明的概念（或标</span> <span class="calibre7">签）。看透它们，抓本质。</span></p><p class="calibre1">15. 硬规则</p><p class="calibre1"><span class="calibre7">世上有些硬规则。拿头撞墙，不会好受。伤害他人，必有后果。丢了信誉，日后难融资、成</span> <span class="calibre7">功。这些都是人生硬规则，尊重它们。</span></p><p class="calibre1"><span class="calibre7">16. </span><span class="calibre22">要</span><span class="calibre22">有全球眼光</span></p><p class="calibre1"><span class="calibre7">币安是全球分布最广的组织之一。以前没哪个组织，能在用户、团队、思维上，到我们这「全</span> <span class="calibre7">球化」程度。</span></p><p class="calibre1"><span class="calibre7">这给了我们别人没有甚至不知道的优势。我们能全球招人；团队天然跨每个时区。我们远程工</span> <span class="calibre7">作，大家不用通勤搞得又累又出汗，每天还省一两小时。我们在 180 多个国家一起发展，既本</span> <span class="calibre7">地化又全球化。好处说不完。</span></p><p class="calibre1"><span class="calibre7">挑战也有。沟通方式不同，后面说。文化不同，别人对世界的理解也不同。要是别人跟你想的</span> <span class="calibre7">不一样，你就不舒服，那币安可能不适合你。要是你喜欢探索不同文化、不同想法，那币安就</span> <span class="calibre7">很适合。</span></p><p class="calibre1"><span class="calibre7">很多人觉得自己开放，能容全球文化，直到跟人吵一架。然后就觉得「对方就是蠢」。小心这</span> <span class="calibre7">个。试着真有全球眼光。可以看《文化地图》。</span></p><p class="calibre1"><span class="calibre7">另外注意，很多标签在不同地方意思不同。比如在中国，一年转出超过 5 万美元等值货币，会</span> <span class="calibre7">被管制。在世界很多地方，娶四个老婆合法正常。这样的例子多了。</span> <span class="calibre7">我们都住在一个叫地球的小石头上。国家边界是人画的。要有全球观。</span></p><p class="calibre1"><span class="calibre10">团队</span></p><p class="calibre1">17. 团队大于个人</p><p class="calibre1"><span class="calibre7">在一个不行的团队里，个人干得再好，也难满意。反过来常成立。团队行，每个成员，哪怕干</span> <span class="calibre7">得吃力的，也通常不差。</span></p><p class="calibre1"><span class="calibre7">团队利益该优先，对每个成员长期也是最好的。短期你可能得「为团队牺牲一下」，但长期你</span> <span class="calibre7">赢。</span></p><p class="calibre1">18. 常轮换团队</p><p class="calibre1"><span class="calibre7">别让组织僵住。给新领导更多成长机会。快调「放错位置的人」（这也可能带来更多问题，</span> <span class="calibre7">但……）。</span></p><p class="calibre1"><span class="calibre7">团队结构决定了系统架构。可以看《团队拓扑学》。我们不希望软件僵住，团队结构也得常</span> <span class="calibre7">调。</span></p><p class="calibre1">19. 内部竞争不是坏事</p><p class="calibre1"><span class="calibre7">外面竞争永远在，内部有点竞争是好事。只要保持专业就行。</span> 20. 有序的混乱也是一种秩序</p><p class="calibre1"><span class="calibre7">这有点反直觉，不好说。看「混乱」和「有序」两个极端。全乱肯定不行，这好懂。</span> <span class="calibre7">很多人觉得组织越有秩序越好。我不同意。秩序清楚好处多，权责分明，少重叠、少浪费，通</span> <span class="calibre7">常更高效。</span></p><p id="page_230" class="calibre1"><span class="calibre7">但多数人没想的是：秩序通常只能让组织高效干一件事。过于有秩序也坏事。</span> <span class="calibre7">世界在变，结构太死板的组织要费更多劲去适应。我们这行还年轻，世界变得快。结构僵化的</span> <span class="calibre7">组织，创新少，内部竞争少，晋升机会也少。</span></p><p class="calibre1"><span class="calibre7">这不是说「混乱」一定好。平衡点在哪？我们活在一个变动的世界，新兴行业。我们这行重定</span> <span class="calibre7">义了传统概念：总部、公司、团队，甚至钱。我们远程全球办公，很多传统结构不适合我们。</span> <span class="calibre7">同时，我们在很多方面又得强管。我们处理用户的钱。安全、合规、道德、中立，都得严管。</span> <span class="calibre7">我们在受监管环境里运营，合规特重要。</span></p><p class="calibre1"><span class="calibre7">一个「有序的混乱」环境，需要最好的人才，有热情、有责任心的人。怎么找这个平衡是持续</span> <span class="calibre7">的挑战。关键是，混乱有时候也是一种特色。</span></p><p class="calibre1">21. 多做本地团建</p><p class="calibre1"><span class="calibre7">争取一月一次。但实际，因为安排之类，通常两月一次。</span> <span class="calibre7">一起吃顿饭是最好的团建。简单又管用。</span></p><p class="calibre1"><span class="calibre7">就在本地搞，跨团队搞。我不赞成为了团建飞来飞去。差旅成本高，时间成本也太大。我不想</span> <span class="calibre7">让人觉得我们就是带人飞来飞去「玩」。对确实分散的小团队，每 18 个月搞一次大型的作为</span> <span class="calibre7">特例可以，最好能结合已有的出差或活动。</span></p><p class="calibre1">22. 给反馈</p><p class="calibre1"><span class="calibre7">我随时随地直接给反馈，私聊或大群。这是从 Netflix 那本书（《不拘一格》）学的。</span> <span class="calibre7">其实我更喜欢在大群给反馈，这样别人也能学到，我不用重复多次。</span> <span class="calibre7">很多人第一次收到这种反馈会惊一下，但很快就习惯了。我给反馈都是好意，为了解决问题或</span> <span class="calibre7">把事情做得更好。</span></p><p class="calibre1"><span class="calibre7">我想在币安建一种坦诚反馈的文化。我觉得 99% 的人给别人反馈都不够。远程工作，我们看不</span> <span class="calibre7">到线下会里的肢体语言。所以必须靠直接、坦诚的反馈来补。</span> <span class="calibre7">桥水基金 (Ray Dalio) 有个 DOT 反馈系统，我很喜欢。以后我们会想法子用上。</span> 23. 口头表扬不多</p><p class="calibre1"><span class="calibre7">你要是干得好，可能听别人说「干得好，太棒了」。我可能说得少。反过来，干得差，我看见</span> <span class="calibre7">了，十有八九你会听到我叨叨。</span></p><p class="calibre1"><span class="calibre7">我的理由是：</span></p><p class="calibre1"><span class="calibre7">1.我们标准高，追求卓越，好结果该是常态。每个「小」成就都大张旗鼓，好像我们在设低标</span> <span class="calibre7">准。</span></p><p class="calibre1"><span class="calibre7">2.远程，我看不到所有团队的成绩，没法面面俱到。我只表扬一些，没表扬另一些，容易心理</span> <span class="calibre7">不平衡，或让人觉得我有偏向。</span></p><p class="calibre1"><span class="calibre7">3.满足感该从心里来。我干好一件事，内心奖励已足够。</span> <span class="calibre7">4.不高效。事干完了，就该盯下一个。</span></p><p class="calibre1"><span class="calibre7">我不是说这是好方法，这是我的习惯。我觉得「胡萝卜加大棒」更好，但我还没想在分散环境</span> <span class="calibre7">里怎么高效给「胡萝卜」。</span></p><p class="calibre1"><span class="calibre7">不同文化对这块期望也不同。《文化地图》讲得好。美国人通常是「干得真棒」文化，亚洲人</span> <span class="calibre7">更多是「大棒」文化。欧洲人在两者之间。书里说的。</span> <span class="calibre7">更高效的是下次考核时调薪酬。所以我也会给「胡萝卜」，就是嘴上给得少。</span> <span class="calibre7">自打发了第一版，我收到不少反馈说这样不好，我也不反对。</span></p><p id="page_231" class="calibre1">24. 上报问题 vs 传闲话</p><p class="calibre1"><span class="calibre7">上报问题是正常的，但有对错方式。</span></p><p class="calibre1"><span class="calibre7">传闲话是你私下跟我 1 对 1 抱怨别人，但没告诉当事人。传闲话不好。我不处理闲话，直接</span> <span class="calibre7">忽略。实际上，你这么做，我心里给你记个差评（不是给被抱怨的人）。</span> <span class="calibre7">上报问题是你约个三人会，我、你，还有你想抱怨的人。这样我能一次听两边说法。</span> <span class="calibre7">更重要的是，这迫使你在找我之前，先跟对方坦诚沟通一次。</span> <span class="calibre7">我只需要开一次会处理上报。相比处理闲话，我得开多次一对一。你知道我对时间的态度（后</span> <span class="calibre7">面说）。</span></p><p class="calibre1"><span class="calibre7">用上报，别传闲话。这难，但能用专业方式表达你对某人或某事不满，是成功的关键技能之</span> <span class="calibre7">一。别太软，别太硬，找对平衡是关键。</span></p><p class="calibre1"><span class="calibre7">25. </span><span class="calibre22">招聘</span></p><p class="calibre1"><span class="calibre7">团队对成功特重要。实际上，对多数成功人士，他们最大贡献就是吸引一群才华横溢、有热情</span> <span class="calibre7">的人，让他们高效协作。我觉得这就是我对币安最大的价值。</span> <span class="calibre7">招聘在这过程里显然很重要。</span></p><p class="calibre1"><span class="calibre7">招有热情的人</span></p><p class="calibre1"><span class="calibre7">热情是我最看重的之一。我们远程，没法（也不该）微观管理。没热情的人会偷懒，然后被淘</span> <span class="calibre7">汰掉。要雇那个想盖教堂的人。</span></p><p class="calibre1"><span class="calibre7">使命比钱重要</span></p><p class="calibre1"><span class="calibre7">别雇对薪水和待遇斤斤计较的人。这种人很少是使命驱动的。薪酬该给得公平。</span> <span class="calibre7">在招聘阶段就过分讨价还价的人，通常也是「高维护型」的人。你知道我对这类人的看法。</span> <span class="calibre7">招有饥饿感的人</span></p><p class="calibre1"><span class="calibre7">招能在岗位上成长的人，不是那种「干过、懂」的人。虽然过往经验在很多情况有帮助，在合</span> <span class="calibre7">规等某些岗位甚至是前提，但后者容易无聊，导致「定式思维」，容易被过往经验框住。</span> <span class="calibre7">招实干家，不招空谈家</span></p><p class="calibre1"><span class="calibre7">招能干也能表达的人，不是只会说不会干的。能干但不会表达的实干家麻烦，在窄的技术领域</span> <span class="calibre7">可能还行，但我们团队里这种人也不能多。</span></p><p class="calibre1"><span class="calibre7">别在意头衔</span></p><p class="calibre1"><span class="calibre7">别招特别在意头衔的人。不会一票否决，但肯定不是好信号。</span> <span class="calibre7">招能当你上司的人</span></p><p class="calibre1"><span class="calibre7">永远招最好的人。要赢，你得是强队的一员。招比你强的、和你差不多的、甚至能做你上司</span> <span class="calibre7">的。招个能做你上司的人，是职业生涯进步的最好方式之一，也说明你成熟度高。</span> <span class="calibre7">有疑虑，就不招</span></p><p class="calibre1"><span class="calibre7">如果招聘过程中你有疑虑，就别招。面试时的小疑虑，将来总会变大事。</span> 26. 别试图激励没自驱力的人</p><p class="calibre1"><span class="calibre7">像拖死马，不可能，也不值。那些跟你使命、价值观不合，不喜欢你这个领导，或者就是懒的</span> <span class="calibre7">人，你没法激励他们。让他们去别处干。人要么有自驱，要么没有。只跟有自驱的人共事。</span> <span class="calibre7">我们远程，懒人容易摸鱼，因为没人盯着。但这是好事。人可以摸一天、一周，甚至一月，但</span> <span class="calibre7">几月后，拿不出结果，你就知道了。远程反而容易让你发现他们。一旦发现，让他们尽快走</span> <span class="calibre7">人。</span></p><p class="calibre1"><span class="calibre7">我不搞动员演讲。我觉得那只是短暂刺激和驱动下，不长久。</span> 27. 以身作则</p><p id="page_232" class="calibre1"><span class="calibre7">努力工作，坚守价值观，以身作则。人们看你怎么做，不是听你怎么说。</span> 28. 永远不要微观管理</p><p class="calibre1"><span class="calibre7">微观管理比你自己干还费时。如果你需要微观管理一个人，那就该让他走。</span> 29. 招聘看资历，之后看成果</p><p class="calibre1"><span class="calibre7">用「年限」来筛招聘。一旦入职，就用成果来量表现。</span> <span class="calibre7">30. </span><span class="calibre22">淘汰末位</span></p><p class="calibre1"><span class="calibre7">我信「末位淘汰」。我看过很多反对强制分布、反对制造内部竞争的说法。我不信。</span> <span class="calibre7">按我经验，高绩效的人喜欢跟高绩效的人共事。一群高绩效的人配合好了，工作本身会上瘾。</span> <span class="calibre7">要是混进个低绩效的，一切都被拖垮。把末位的请走。</span> <span class="calibre7">我也信《不拘一格》那书里说的「团队，不是家庭」。听着不舒服，但组织和家庭确实有点不</span> <span class="calibre7">同。我们彼此关心，但我们不会带低绩效的一起走。这对团队其他成员也不负责。</span> 31. 用产出当目标</p><p class="calibre1"><span class="calibre7">用产出指标（用户、收入、市场份额），不是输入指标（任务、功能、会议、工时）。</span> 32. 别把目标太当回事</p><p class="calibre1"><span class="calibre7">目标和目标设定有很多潜在缺点，很多人写过。我不细说，比如达不到时的挫败感，轻松达到</span> <span class="calibre7">后就不努力了，方向死板等等。</span></p><p class="calibre1"><span class="calibre7">我对目标最大的意见是：</span></p><p class="calibre1"><span class="calibre7">1.它们永远不够精确或科学，基本是猜的。我们这行，市场变得太快。</span> <span class="calibre7">2.讨论目标太花时间（代价高）。</span></p><p class="calibre1"><span class="calibre7">所以，定个目标，朝它努力，达到了就定新目标。别太当回事，别钻进去。</span> <span class="calibre7">举个例子。币安刚起步，我们定了一个目标，三年内成全球前十的交易所。结果，五个月成全</span> <span class="calibre7">球第一。我们依旧没停下。</span></p><p class="calibre1">33. 容忍失败，但不是所有</p><p class="calibre1"><span class="calibre7">你常听「我们容忍失败」，然后又听我们强调「问责制」。这俩啥关系？冲突吗？</span> <span class="calibre7">我觉得问责是任何成功团队的关键。所以复盘时你常听我问「谁」，而且必须是具体人名，不</span> <span class="calibre7">是团队，不是小组。这容易被误会成「不容忍失败」，甚至「甩锅文化」。</span> <span class="calibre7">我对失败看法更细一点。</span></p><p class="calibre1"><span class="calibre7">我能接受某些类型的错误，但不是所有。</span></p><p class="calibre1"><span class="calibre7">我能接受诚实、努力、承认的错。我很少罚为诚实错误担责的人。</span> <span class="calibre7">我不能接受掩盖错、糊弄、推给团队或离职人员的做法。你越盖，我越「挖地三尺」。我也不</span> <span class="calibre7">能接受完全疏忽、不关心、不知情况、什么也不干的人。我不能接受因为懒犯的错。</span> <span class="calibre7">努力干，尽力，透明负责，然后不用怕。</span></p><p class="calibre1"><span class="calibre10">生意与合作</span></p><p class="calibre1">34. 让生意简单点</p><p class="calibre1"><span class="calibre7">变量太多的复杂合作容易黄，签了也难走。复杂合作难懂，容易误会。总有一方觉得在某些方</span> <span class="calibre7">面亏了，想改。让合作简单：甲方给这，拿那；乙方给这，拿那。</span> 35. 早点对无用合作说「不」</p><p id="page_233" class="calibre1"><span class="calibre7">太多人花时间在没用的「合作」讨论上。你的心智被这些占着，就没空思考真有用的合作了。</span> 36. 快速推进，或者放弃</p><p class="calibre1"><span class="calibre7">要么快速推进，要么尽早说不。俩都行。最差的是慢吞吞。</span> <span class="calibre7">想法慢吞吞的机会成本高得离谱。它占着你脑子，让你没法干别的。如果事推太慢，就放。不</span> <span class="calibre7">值当干，往前走。</span></p><p class="calibre1"><span class="calibre7">这尤其适用于生意合作。如果对方推太慢，或者合作太复杂难敲定，最好就停在那，省得以后</span> <span class="calibre7">麻烦。</span></p><p class="calibre1">37. 拒绝独家</p><p class="calibre1"><span class="calibre7">长期的、双赢的合作不需要独家。要求排他的人，通常是对自己竞争力或能提供的长期价值没</span> <span class="calibre7">信心。这种，短期（或一次性）的补偿方案可能更合适。但你知道我的看法，别在短期合作上</span> <span class="calibre7">花太多时间。世界变得太快，别锁死。你预测不了未来。别签独家合同，别锁死我们自己，也</span> <span class="calibre7">别指望锁死别人。</span></p><p class="calibre1">38. 始终要有终止条款</p><p class="calibre1"><span class="calibre7">合同里永远要有终止条款。需要能从非双赢的关系里退出的方式。永远要有选择权。很多人在</span> <span class="calibre7">签合同时只考虑通常情况（通常乐观），这是错的。要考虑最坏情况。这才是合同的作用。</span> 39. 始终有限责任</p><p class="calibre1"><span class="calibre7">永远别签那种可能导致巨大或「无限」责任的合同。签合同的时候，要想极端最坏情况，不是</span> <span class="calibre7">「正常/最好」情况。</span></p><p class="calibre1">40. 没有特例</p><p class="calibre1"><span class="calibre7">永远别给某个客户别人没有的特殊待遇。永远平等对待所有客户。</span> 41. 被动做业务拓展，做能成的合作</p><p class="calibre1"><span class="calibre7">我（CZ）通常对业务拓展比较被动，生活大体也这样。人们常不理解我这点，或者不知道咋运</span> <span class="calibre7">作。</span></p><p class="calibre1"><span class="calibre7">别把这跟热情混淆。我对所做的事非常有热情，但在怎么接触人、建商业合作上，采取被动方</span> <span class="calibre7">式。</span></p><p class="calibre1"><span class="calibre7">我不追那些光鲜亮丽的。做业务，我通常不追大客户或大伙伴。花大量精力教他们懂加密，手</span> <span class="calibre7">把手帮他们走内部法务、董事会审批，转化时间太长。他们还常要不对等的条件，投入产出比</span> <span class="calibre7">低。</span></p><p class="calibre1"><span class="calibre7">相反，我喜欢花时间跟主动找上门的顶级公司合作。他们本来就有意进加密领域，想跟我们合</span> <span class="calibre7">作，我们只需敲定方式和条件，投入产出比高多了。就算他们可能不是苹果或谷歌那样的巨</span> <span class="calibre7">头，只要我们一直积小胜，大伙伴迟早会来找我们，多半是「自然而然」的。</span> <span class="calibre7">其他例子：我不会浪费时间试图说服那些对加密有成见的人，比如沃伦·巴菲特。我选跟想学</span> <span class="calibre7">的人聊，哪怕他们没那么有名。</span></p><p class="calibre1"><span class="calibre7">我不会去对加密持负面态度的国家或地区，我去那些想接纳加密的地方帮他们，哪怕是小国。</span> <span class="calibre7">核心就是，做那些我们能敲定的合作。</span></p><p class="calibre1"><span class="calibre7">别把这跟「短期思维」混。这些容易摘的果子，就算伙伴可能不是全球最知名的十家公司，也</span> <span class="calibre7">还是长期合作。</span></p><p class="calibre1"><span class="calibre7">这做法有几点要注意。</span></p><p class="calibre1"><span class="calibre7">1.我们得够牛，别人才愿意来找我们。幸运的是币安现在有这条件，我们得保持住。在币安成</span> <span class="calibre7">功之前，我就是这心态，但成功后效果当然更好。</span></p><p class="calibre1"><span class="calibre7">2.我们需要善于选。现在找上门的很多，尤其币安今天这地位。选出最好的请求并不像看起来</span> <span class="calibre7">那么容易。同样，快速抓住合作核心是我常用的方法。</span> <span class="calibre7">3.说了这么多，我们有时还是需要主动出击，以防对方也是「被动」心态。主动联系时要具</span> <span class="calibre7">体、明确，如果对方没回应，那我们就知道他们不感兴趣。</span> <span class="calibre7">生活中也是这样，我不会试图去见某某名人，我跟主动来找我的人互动。</span></p><p id="page_234" class="calibre1"><span class="calibre10">沟通</span></p><p class="calibre1">42. 尽早并常说「不」，省时间</p><p class="calibre1"><span class="calibre7">省时间最有效的工具就是说「不」。</span></p><p class="calibre1"><span class="calibre7">有人想讨论一个「重要」但模糊的合作，我说不。有人请我去见某个大人物，但没明确目的，</span> <span class="calibre7">我说不。有人请我去看艺术展，我说不。有人请我看 F1，我说不。足球赛，不⋯⋯我也会和朋</span> <span class="calibre7">友去这些活动。但默认答案是不。</span></p><p class="calibre1"><span class="calibre7">这样，我能省下时间去做更重要的事，哪怕只是在酒店房间待着。我得以思考和专注于更重要</span> <span class="calibre7">的事，比如写这篇文章。</span></p><p class="calibre1">43. 沟通要简洁直接</p><p class="calibre1"><span class="calibre7">永远把你的意图或目标说清楚。你想要啥？在开始解释背景前，先说「我想要⋯⋯」。对方可</span> <span class="calibre7">能直接就同意了，你就不用再解释了。</span></p><p class="calibre1">44. 写作要简洁</p><p class="calibre1"><span class="calibre7">读《写作法宝》。我讨厌看到人要么不写，要么写太长。这意味着他们要么没花时间，要么没</span> <span class="calibre7">能力理清思路。</span></p><p class="calibre1"><span class="calibre7">对我来说，我不希望看到超过：</span></p><p class="calibre19"><span class="calibre7">●         15 分钟的会，用 3-5 个要点</span></p><p class="calibre19"><span class="calibre7">●         30-60 分钟的团队会，用半页到一页</span></p><p class="calibre19"><span class="calibre7">●         月度或季度业务回顾，最多 5 页</span></p><p class="calibre19"><span class="calibre7">●         不要 PPT ，不要花哨的幻灯片，只要文字和简单的柱状图。</span> <span class="calibre7">●         博客、文章或书，可以写长点。</span></p><p class="calibre1"><span class="calibre7">学会写好。我还在练⋯⋯</span></p><p class="calibre1">45. 消息优先于见面</p><p class="calibre1"><span class="calibre7">有句老话我不认同：「能打电话就别发消息，能见面就别打电话。」我提倡反过来：「能打电</span> <span class="calibre7">话就别见面，能发消息就别打电话。」两种都没错。对于棘手的对话，见面更好。但对大多数</span> <span class="calibre7">日常沟通，我更看重效率而不是形式。</span></p><p class="calibre1"><span class="calibre7">你需要跟你沟通的人有足够强的信任（或信誉），能互相理解，不猜疑，总是往好处想。我们</span> <span class="calibre7">大部分工作都远程，所以我采取这个方式。</span></p><p class="calibre1">46. 避免沟通链条</p><p class="calibre1"><span class="calibre7">别去跟一个听了别人转述的人聊事情。你得到的信息肯定走样。直接找源头聊。</span> <span class="calibre7">工作中，我们常让项目经理或其他领导当中间人。要避免过长的沟通链。把源头拉进一个群或</span> <span class="calibre7">会，同时别让会规模太大。</span></p><p class="calibre1">47. 一条消息说清，别发多条</p><p class="calibre1"><img src="/books/binance-life-memoir/assets/index-234_1.jpg" alt="" class="calibre57" loading="lazy"></p><p id="page_235" class="calibre1"><span class="calibre7">发多条，对方会收到好几个通知，我等回复的时间可能也更长。不如这样：</span></p><p class="calibre1"><img src="/books/binance-life-memoir/assets/index-235_1.jpg" alt="" class="calibre58" loading="lazy"></p><p class="calibre3"> </p><p class="calibre1"><span class="calibre7">一条消息，搞定。</span></p><p class="calibre1"><span class="calibre7">是的，我尽量这样优化我的时间。我不喜欢跟沟通「习惯不好」的人聊。他们时间多，我没</span> <span class="calibre7">有。</span></p><p class="calibre1">48. 别在即时通讯里争论</p><p class="calibre1"><span class="calibre7">不要在聊天里争论或辩论。拿起电话，用视频或语音来辩论。</span> 49. 沟通太多也不好</p><p class="calibre1"><span class="calibre7">沟通太少不好，沟通太多也不好。如果必须不断过度沟通才能推动事情，那说明哪里出问题</span> <span class="calibre7">了。你需要解决根本问题。</span></p><p class="calibre1">50. 提问时带上背景</p><p class="calibre1"><span class="calibre7">我们在远程环境工作，见不到很多同事。问题容易被误解。问问题时，总要说清楚你为什么问</span> <span class="calibre7">这个问题。</span></p><p class="calibre1">51. 带上单位</p><p class="calibre1"><span class="calibre7">不好：「我们花了 40k。」</span></p><p class="calibre1"><span class="calibre7">好：「我们花了 4 万美元。」</span></p><p class="calibre1"><span class="calibre7">不好：「租金是 4 万美元。」</span></p><p class="calibre1"><span class="calibre7">好：「租金是每年 4 万美元。」</span></p><p class="calibre1"><span class="calibre7">别不写单位，指望对方猜对。世界上有很多种货币，现在还有加密货币。沟通中尽量精准。但</span> <span class="calibre7">这块信息可以全一些。</span></p><p class="calibre1">52. 会要短</p><p class="calibre1"><span class="calibre7">会要尽可能短，5 分钟最好。如果你和紧密合作的同事做不到 5 分钟会，说明你们还没找到默</span> <span class="calibre7">契。想办法做到。</span></p><p class="calibre1">53. 准时开会</p><p class="calibre1"><span class="calibre7">提前 1 分钟进会。把闹钟设成 3:59 而不是 4:00，这样就能 4:00:00 准时开始。</span> 54. 不要介绍，不要背景，直接说</p><p class="calibre19"><span class="calibre7">●         别用那种老套的「我先告诉你我要讲什么，然后讲，最后总结我讲了什么」。</span></p><p class="calibre34"><span class="calibre7">○         直接讲内容就行。</span></p><p class="calibre19"><span class="calibre7">●         别以「这里是今天的议程⋯⋯」开头。</span></p><p class="calibre34"><span class="calibre7">○         直接进会。</span></p><p class="calibre19"><span class="calibre7">●         别问「能听到吗？能看到我屏幕吗？」</span></p><p class="calibre34"><span class="calibre7">○         提前测试好设备，直接进会。</span></p><p class="calibre19"><span class="calibre7">●         别「感谢大家参加⋯⋯」</span></p><p class="calibre34"><span class="calibre7">○         直接进会。</span></p><p class="calibre1"><span class="calibre7">用「我/我们想要⋯⋯」开头。比如，我们想要再招 10 个人。我们需要 X 预算。如果对方答</span> <span class="calibre7">「行」，那会就结束了。不需要讲那么多背景和介绍。</span> 55. 讨论人数少于 10 人</p><p class="calibre1"><span class="calibre7">讨论应该只让 5-10 个最懂这主题的人参与。人多了反而慢。</span></p><p id="page_236" class="calibre1"><span class="calibre7">超过 10 人的会，应该是快速同步信息的短会。</span></p><p class="calibre1">56. 移除不说话的人</p><p class="calibre1"><span class="calibre7">如果你参加一个讨论会，全程没说一句话，那你可能就不该参加这个会。你也许看看会议纪要</span> <span class="calibre7">就够了。</span></p><p class="calibre1">57. 会前写好要点</p><p class="calibre1"><span class="calibre7">会前写好要点。写下来能理清思路。我（CZ）是视觉型的人，光靠嘴说，我能记住的很少。书</span> <span class="calibre7">面文件也更容易转发。口头说没法转发。可以搜一下「传话游戏」。</span> <span class="calibre7">书面文件别太长。30 分钟的会，一页纸够了。</span></p><p class="calibre1"><span class="calibre7">58. </span><span class="calibre22">不要</span><span class="calibre7"> PPT</span></p><p class="calibre1"><span class="calibre7">那是浪费时间。用要点和柱状图展示历史和趋势就好。</span> 59. 不要「介绍性」会</p><p class="calibre1"><span class="calibre7">我不参加那些只是见面认识一下、互相了解、探索性的会。我不是枢纽型人，也不擅长维护很</span> <span class="calibre7">多关系。我更喜欢有明确目的的会。有人可能觉得这太「功利」，但这高效。可能会得罪一些</span> <span class="calibre7">人，但我的目标不是跟所有人做朋友，是把事情做成。</span></p><p class="calibre1"><span class="calibre10">产品</span></p><p class="calibre1">60. 专注于用户</p><p class="calibre1"><span class="calibre7">拥有用户是关键，其他都没那么重要。没用户就没价值。好好待用户。</span> 61. 只做能扩大的产品</p><p class="calibre1"><span class="calibre7">只做能规模化的产品。先做最小可行产品，然后要么关，要么转，要么全力扩大。如果不能规</span> <span class="calibre7">模化，就别做。</span></p><p class="calibre1">62. 人人都是产品经理</p><p class="calibre1"><span class="calibre7">每个人都是产品经理。只要你在工作，你就是产品经理。生活中，你是自己的产品经理。产品</span> <span class="calibre7">开发能写很多，我不展开了。</span></p><p class="calibre1"><span class="calibre10">公共关系</span></p><p class="calibre1">63. 不要搞大动静的发布</p><p class="calibre1"><span class="calibre7">上线第一天别搞大宣传。总会出问题。等系统/产品稳定跑一周左右，再做大宣传。</span> 64. 不要宣传空洞的备忘录或意向书</p><p class="calibre1"><span class="calibre7">只宣传有结果的事。除非宣传本身能给我们带来显著好处。要小心那些只想借我们品牌给自己</span> <span class="calibre7">背书的小合作方。</span></p><p class="calibre1">65. 不延迟发布，好了就宣布</p><p class="calibre1"><span class="calibre7">我不认同的一种做法：有时候，公关团队建议等某个日期或时间再宣布已经准备好的事。理由</span> <span class="calibre7">可能包括：现在是周五晚上，新闻效果会差，等周一早上；我们刚宣布了别的事，需要稍微错</span> <span class="calibre7">开；留着等圣诞节期间再发，因为那时新闻少。这些只会造成不必要的拖延，把所有后续工作</span> <span class="calibre7">都往后推，效率的损失远远大于「新闻效果更好」那点微不足道的优化。我们在工作上的拖延</span> <span class="calibre7">成本极高。推迟发布只是让这件事在人们脑子里多停留一段时间，毫无必要。</span> <span class="calibre7">好了就宣布，然后去做下一件事。</span></p><p id="page_237" class="calibre1"><span class="calibre7">发推文也一样。很多社交媒体专家建议在一天中最佳时间发推。如果你专门做社交媒体，这或</span> <span class="calibre7">许有用。我脑子里有想法就发。否则，让想法在脑子里多停留。</span></p><p class="calibre3"> </p><p class="calibre1">65. 不延迟发布，好了就宣布（续）</p><p class="calibre1"><span class="calibre7">我不认同的一种做法：有时候，公关团队建议等某个日期或时间再宣布已经准备好的事。理由</span> <span class="calibre7">可能包括：现在是周五晚上，新闻效果会差，等周一早上；我们刚宣布了别的事，需要稍微错</span> <span class="calibre7">开；留著等圣诞节期间再发，因为那时新闻少。这些只会造成不必要的拖延，把所有后续工作</span> <span class="calibre7">都往后推，效率的损失远远大于「新闻效果更好」那点微不足道的优化。我们在工作上的拖延</span> <span class="calibre7">成本极高。推迟发布只是让这件事在人们脑子里多停留一段时间，毫无必要。</span> <span class="calibre7">好了就宣布，然后去做下一件事。</span></p><p class="calibre1"><span class="calibre7">发推文也一样。很多社交媒体专家建议在一天中最佳时间发推。如果你专门做社交媒体，这或</span> <span class="calibre7">许有用。我脑子里有想法就发。否则，让想法在脑子里多待几小时带来的心理成本，不值得那</span> <span class="calibre7">点所谓的最佳效果。发完，我就去做别的事了。</span></p><p class="calibre1">66. 回应记者</p><p class="calibre1"><span class="calibre7">如果不回应，他们就会按最坏的情况来写。回复他们，记下来，必要时可以自己发。</span> 67. 快速回应负面新闻</p><p class="calibre1"><span class="calibre7">否则负面新闻会扩散。除非你确定那是家小媒体，掀不起风浪。</span> 68. 决策框架</p><p class="calibre1"><span class="calibre7">做出好决策是能练习的。我大致按这个思维框架来。</span></p><p class="calibre1"><span class="calibre7">第一性原理</span></p><p class="calibre1"><span class="calibre7">如果决策触及核心原则，就按原则来。简单。</span></p><p class="calibre1"><span class="calibre7">小事 vs 大事</span></p><p class="calibre1"><span class="calibre7">小事，快速决定，或者授权给别人，然后继续。比如去哪吃饭，或者一笔小投资。影响小的</span> <span class="calibre7">事。</span></p><p class="calibre1"><span class="calibre7">大事，收集数据，小组讨论，然后过一晚上再定。比如大的赞助、投资等。</span> <span class="calibre7">可逆 vs 不可逆</span></p><p class="calibre1"><span class="calibre7">有些决定是可逆的，比如开发一个新功能。随时可以停，转去做别的，或者做完了再关掉。会</span> <span class="calibre7">损失一些时间和精力，但有限。这些可以快速决定，试错有预期。</span> <span class="calibre7">有些决定不容易逆，比如一笔大额赞助的预付款，收购一家大公司并计划整合团队等。这些需</span> <span class="calibre7">要更慎重。</span></p><p class="calibre1"><span class="calibre7">我是否专业？</span></p><p class="calibre1"><span class="calibre7">如果是自己熟悉、有背景的领域（比如技术、产品），我决策会快些。在我没底的领域，比如</span> <span class="calibre7">市场营销，要么授权，要么请教专家，或者更谨慎地决定。</span> <span class="calibre7">信息是否够？</span></p><p class="calibre1"><span class="calibre7">最后问自己，我/我们信息够吗？小事不需要太多信息。大事，应该尽量拿必要信息。但归根结</span> <span class="calibre7">底，我们常常要在信息有限的情况下做决定。</span></p><p class="calibre1"><span class="calibre7">最后：通常做个决定并执行，比不做决定好。不做决定本身就是一个决定。</span></p><p class="calibre1"><span class="calibre10">休息、保持冷静、放松</span></p><p class="calibre1"><span class="calibre7">有人问我睡多少小时、怎么倒时差之类的，所以我加了这块。</span> <span class="calibre7">69. </span><span class="calibre22">睡眠</span></p><p class="calibre1"><span class="calibre7">建议找到能让你精力最充沛的睡法。</span></p><p class="calibre1"><span class="calibre7">我晚上睡 5-6 小时，下午通常再小睡 30-45 分钟。小睡后我精神最好，其次是早上醒来一小</span> <span class="calibre7">时后。我会在这个时间段做高强度的思考或艰难的决定。其余时间处理「日常」事。</span> <span class="calibre7">一个小秘密。小睡也是我倒时差的方法。时差严重时，我会睡久点。有两段睡眠帮助很大。</span> <span class="calibre7">另外，累了我就放松或小睡。</span></p><p id="page_238" class="calibre1">70. 性格冷静</p><p class="calibre1"><span class="calibre7">我性格冷静。如果别人情绪波动幅度是 10，我大概是 5。我也有强烈的情绪，但不会过度兴奋</span> <span class="calibre7">或悲伤。这种冷静的性格在高压情况下很有帮助，我们作为新兴快节奏行业的创业公司，常见</span> <span class="calibre7">面这种情况。</span></p><p class="calibre1"><span class="calibre7">这种性格一部分天生，一部分后天练的。我信「模拟理论」，这对保持情绪冷静很有帮助。</span> <span class="calibre7">坚守高道德标准也有帮助。知道自己在做对的事，就没太多可担心的。这让我心理强大。知道</span> <span class="calibre7">自己对世界有积极影响，也有帮助。</span></p><p class="calibre1">71. 不追求豪华办公室</p><p class="calibre1"><span class="calibre7">这点可能也有争议。我不信豪华办公室那套。我在彭博办公室干过，有鱼缸、无限量食物、按</span> <span class="calibre7">摩服务。我不觉得那些能提高生产力或士气。我觉得这和住一个风景很好的公寓一样，两秒钟</span> <span class="calibre7">就习惯了，之后没任何作用。</span></p><p class="calibre1"><span class="calibre7">我更喜欢一个功能齐全的（家里或酒店）「办公室」，有高速网络、外接显示器、升降桌和手</span> <span class="calibre7">机支架。旅行时，我把行李箱放桌上当升降桌。我喜欢用手机通话，同时在笔记本上看文档。</span> <span
